This is a link list of free RDF learning resources for beginners.
It is also related to topics like the Semantic Web and Knowledge Graphs.

We used Twitter and the hashtag [#learnRDF](https://twitter.com/search?q=%23learnRDF&src=typed_query) to collect the resources.
If you are aware of additional high-quality resources, please feel free to use the hashtag and/or reply to the original tweet [🐣](https://twitter.com/DiceResearch/status/1473292972334755840). Share your knowledge and let the community grow.


## Articles & Introductions

- [RDF 1.1 Primer](https://w3.org/TR/rdf11-primer/) of the World Wide Web Consortium (W3C)  
  Contents: Resource Description Framework (RDF), Triples, Vocabularies, serializations (e.g. Turtle, JSON-LD, RDF/XML).
  [🐣](https://twitter.com/DiceResearch/status/1473293240149368835)
- [RDF 1.1 Concepts and Abstract Syntax](https://w3.org/TR/rdf11-concepts/) of the World Wide Web Consortium (W3C)  
  Contents: Similar to the primer above; Graphs, triples, namespaces, datatypes.
  [🐣](https://twitter.com/DiceResearch/status/1473293240149368835)
- [Semantic Web Technologies and Knowledge Graphs](https://github.com/turing-knowledge-graphs/teaching/tree/main/city)  
  is a course of the Alan Turing Institute. It contains slides and exercises with solutions in Java and Python.
  [🐣](https://twitter.com/ejimenez_ruiz/status/1473963637895319556)


## Videos of lectures

- [Information Service Engineering (ISE) lecture at YouTube](https://www.youtube.com/c/ISEFIZKarlsruhe/playlists?view=50&sort=dd&shelf_id=2)  
  Each single lecture comes with a related playlist. In the video comments you'll find links to the slides used in the videos.
  [🐣](https://twitter.com/adrianwilke/status/1473294143216623617)
- [Knowledge Graphs lecture videos at openHPI](https://open.hpi.de/courses/knowledgegraphs2020)  
  Contents: Knowledge representation, Semantic Web, RDF, SPARQL, OWL, Knowledge Graphs, Applications.
  [🐣](https://twitter.com/adrianwilke/status/1473294143216623617)
- [Harald Sack at openHPI](https://open.hpi.de/courses?q=Harald%20Sack)  
  Lectures with overlapping contents from different years: Knowledge Graphs, Information Service Engineering, Linked Data Engineering, Semantic Web Technologies.
  [🐣](https://twitter.com/adrianwilke/status/1473294143216623617)


## Related to programming languages

- [Apache Jena](https://jena.apache.org/tutorials/rdf_api.html)  
  Jena is a Java RDF Framework. The packages and dependencies are available via [Maven](https://mvnrepository.com/artifact/org.apache.jena).
  [🐣](https://twitter.com/UQudus/status/1481582218082459650)
- [A Gateway to Knowledge Graphs](https://ldtim-kgwithr.netlify.app/)  
  R language, using the [rdflib](https://cran.r-project.org/web/packages/rdflib/) R package.
  [🐣](https://twitter.com/NovasTaylor/status/1478489804220383240)
